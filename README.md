# React Hello World

This is the vite react (js, not typescript) template without the vite boilerplate.
It could be useful to start a new project without the need to remove the vite boilerplate.

## Getting Started

To get started, clone the repository and install the dependencies:

```bash
git clone https://github.com/emanuelefavero/react-hello-world.git
cd react-hello-world
npm install
npm run dev
```

This will start the development server and you can view the application in your browser at `http://localhost:5173`.
